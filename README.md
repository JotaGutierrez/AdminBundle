# LIN3S Admin Bundle

> A foundation to create your own admin panel

## Requirements

* PHP >= 5.5
* Symfony >= 2.8 

## Installation
The easiest way to install this component is using **[Composer][1]**

```bash
$ composer require lin3s/admin-bundle
```
